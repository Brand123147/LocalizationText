UGUILocalizationText is a multi-language control.In the game you can switch language  immediately.
You can managed different languages  in the same text.It only needs one code to switch the whole game language.
This tool have KeyString field and Preview field.You can preview the language which you set on inspector panel. 
You can set KeyString field on inspector or on your code.
You can edit text in Localization.txt which is the path:Assets/Scripts/Localization/Localization.txt.
When you initialized language in your code the UnityEditor will remember this language as the first language.
In Resources folder you can change font style in FontStyle prefab.You can use it to change fonts in batches.
